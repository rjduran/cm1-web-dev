hello-live-server

I can preview this page on my localhost with live-server

I can run a local-server instance in any project directory in my sandbox, I could open multiple instances in different terminal windows and specify different ports to access them on. For example, live-server --port=8081 and live-server --port=8082 where instance 1 is running on port 8081 and instance 2 is running on port 8082 and available in different browser windows. Open two terminal windows and try it!

  • Instance 1: http://127.0.0.1:8081/
  • Instance 2: http://127.0.0.1:8082/
